What is Programming?
- A program is a set of instructions that express how to solve a problem, or perform a written task 
- Its written in a specialised language that can be executed by a computer 
- Can be read and understood by other programmers 

 

There are 3 main components: 
  - Sequence;  Steps are carried out in order, one at a time 
  - Selection; Some steps are included, or omitted depending on some conditions 
  - Iteration; Some steps are repeated until something happens
