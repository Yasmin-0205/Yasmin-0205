* Why Programming?

Programming is a skill 
 

A Developers Role  primary role is to create software applications by writing, testing and maintaining/executing code 
Collaboration, problem-solving, and communication are all the key skills required for a successful developer 

 
An analyst / designers primary role is to understand the user requirements and effectively communicate that to the developer 


An architects primary role is to design the overall structure and framework of software systems 
This involves making high level decisions about technology stacks, system components and integration strategies 


A system administrator's primary role is to manage and maintain computer systems and networks 
This involves installing software, monitoring performance and troubleshooting issues 

 
A security specialists primary role is to protect an organisation's computer systems and networks from cyber threats  

Learn to use debugging tools and profilers. The IDE's make life a lot easier 
Get used to the currect collaboration tools like Teams and Slack.
